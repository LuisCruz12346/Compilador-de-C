# Compilador de C
## CRUZ HERNANDEZ LUIS ENRIQUE

Los archivos es el desarrollo lexico de una expresion, por medio de la identificacion de sus tokens, representado en el archivo
en el archivo main.cpp, por otra parte se encuentra el archivo sintactico.cpp, que hace referencia al analisis sintactico de 
una expresion en C. 

El analisis realizado se reconoce el algorito SLR como alternativa robusta para la compilacion de un archivo en C con la gramatica
elegida, con la resolucion de los problemas de recursividad a la izquierda se puede reconocer que este tipo de algoritmo podria 
modelar todo el lenguaje C, pero su automata quedaria excesivamente extenso, sin embargo, el analisis de las expresiones
modelan el claro punto que el lenguaje podria ser modelado por medio del algoritmo.
